# Zomato Project Overview
The Zomato project involves analyzing restaurant data to understand customer preferences, ordering patterns, and spending habits. This includes examining different restaurant types, their ratings, voting patterns, online/offline ordering trends, and cost analysis to help Zomato improve its services and marketing strategies.

# Project Objectives
1.	Understand customer ordering patterns and preferences
2.	Identify which restaurant types are most popular
3.	Analyze rating trends across different restaurant categories
4.	Determine average spending habits
5.	Compare online vs. offline ordering performance
6.	Provide data-driven recommendations to improve Zomato's services
